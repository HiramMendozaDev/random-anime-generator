Random Anime Generator

This project fetches and displays information about a random anime using the Jikan API, which provides data from MyAnimeList. It selects a random anime ID, retrieves its full details, and presents key information such as title, synopsis, genres, score, and more. The project is developed with vanilla JavaScript and styled using the SimpleCSS framework.

Note: Some results may include NSFW tags.
